# Dynatrace OSS Homebrew Tap

Homebrew formulae for Dynatrace open-source tools.

## Available Formulae

| Formula | Description |
|---------|-------------|
| [dtctl](https://github.com/dynatrace-oss/dtctl) | kubectl-inspired CLI for Dynatrace |

## Installation

```bash
brew install dynatrace-oss/tap/dtctl
```

## Updating

```bash
brew update
brew upgrade dtctl
```
